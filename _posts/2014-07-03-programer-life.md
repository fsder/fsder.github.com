```go

func Program() {
    for {
        Think()
        Code()
        Test()
    }
}

```

相信程序员都能看懂这段这程序(golang) 所表达的意思, 这段程序清晰, 完整, 甚至美观，无需注释,  除了没有实际用处.  
那程序可不可以用来写文章?  
有没有人用程序来写诗?  

其实对于一个程序员，能够将程序写的清晰，可阅读就已经很了不起了。
